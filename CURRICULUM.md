## A 数と計算（計算内容の要約）

### 1年生
- 数の意味・表し方：個数を比べる/数える、数の系列・数直線、20までの数、2位数の理解、簡単な3位数、十を単位
- **加法・減法：1位数＋1位数 と その逆の減法（くり上がり・くり下がりを含む）が確実にできる**。簡単な場合の2位数±1位数
- 用語・記号：一の位、十の位、＋、−、＝

### 2年生
- 数：4位数までの十進位取り、十・百を単位、数を積でみる、**1/2・1/3 など簡単な分数を「知る」**
- **加法・減法：2位数の加減（＝筆算）。簡単な場合は3位数などの加減**
- **乗法：乗法九九（1位数×1位数）。簡単な場合は2位数×1位数**
- 用語・記号：直線、直角、頂点、辺、面、単位、×、＞、＜

### 3年生
- 数：万の単位（1億も扱う）、10/100/1000倍・1/10
- **加法・減法：3位数や4位数の加減（筆算）**
- **乗法：2〜3位数 ×（1〜2位数）の乗法（筆算）**
- **除法（わり算）導入：意味・あまり、除数と商が共に1位数の除法が確実、簡単な場合は除数1位数・商2位数**
- 小数：1/10の位、小数の加減 ／ 分数：単位分数、簡単な分数の加減
- □を使った式、そろばん
- 用語・記号：等号、不等号、小数点、1/10の位、数直線、分母、分子、÷

### 4年生
- 数：億・兆、概数（四捨五入・見積もり）
- **整数の除法（筆算）：除数1〜2位数で被除数2〜3位数。被除数＝除数×商＋あまり**
- **小数：小数×整数・小数÷整数、小数の加減** ／ **分数：同分母の加減、大きさの等しい分数**
- 四則混合・( )・公式・□△の式、計算のきまり、そろばん
- （C 変化と関係：折れ線グラフ、簡単な割合）

### 5年生
- 数：偶数・奇数、約数・倍数（公約数・公倍数）、小数点移動
- **小数：小数×小数・小数÷小数**
- **分数：整数÷整数＝分数、約分・通分、相等・大小、異分母分数の加減**
- （C：比例、単位量あたり・速さ、割合・百分率）

### 6年生
- **分数：分数×分数・分数÷分数**
- **文字を使った式（a, x）**
- （C：比例・反比例、比 ／ D：代表値・場合の数）

## 本アプリの網羅状況（A 数と計算）

| 学年 | カリキュラム上の計算 | 実装 |
|---|---|---|
| 1年 | たし算（くり上がり有/無） | ✅ `tashizan`（add10/carry1/mix20） |
| 1年 | ひき算（くり下がり有/無） | ✅ `hikizan`（s10n/sb/s20mix） |
| 2年 | たし算の筆算（2けた） | ✅ `tashizan`（d2p1n/c, d2p2n/c） |
| 2年 | ひき算の筆算（2けた） | ✅ `hikizan`（d2m1n/b, d2m2n/b） |
| 2年 | **かけ算（九九）** | ❌ 未実装（最優先・2年の最重要単元） |
| 3年 | 3〜4けたの加減（筆算） | ✅ `tashizan` d3p3 ／ `hikizan` d3m3（3けた） |
| 3年 | かけ算筆算／わり算（あまり）／小数・分数 | ❌ 未実装 |
| 4〜6年 | わり算筆算／小数×÷／分数×÷／文字式 ほか | ❌ 未実装（トップに枠だけ用意・🔜） |

### 結論（網羅性チェック）
- たし算・ひき算は1〜3年の範囲を **くり上がり/くり下がりの有無で刻んで** カバー済み。
- **唯一の大きな欠落は「九九（かけ算）」**（2年の山場）。次の実装の最優先。
- 4年以降の計算（わり算筆算・小数/分数の四則・文字式）は枠を用意し、順次追加していく。
