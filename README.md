# さんすうクイズ（小学1〜2年むけ）

漢字クイズ（`~/kanji-quiz`）の UI/UX・育成システムを参考に作った、**たし算・ひき算の自動生成クイズ**。
ブラウザで開くだけ（`file://` でも GitHub Pages でも動く）。スマホ・タブレット・PC 対応。

## あそぶ
- トップ（メニュー）: `index.html`
- たし算クイズ: `tashizan/index.html`（テンキー入力／くり上がりヒント付き）

## 構成（漢字クイズと同じ「ポータル＋共有モジュール」方式）
```
/index.html              … トップ（メニュー＝ポータル）
/tashizan/index.html     … たし算クイズ本体（テンキー入力）
/shared/
  sansu-ui.css           … 全アプリ共通デザイン（漢字版 kanji-ui.css ＋ テンキー追加分）
  sansu-core.js          … 育成エンジン SansuCore（カード図鑑・称号17段・メダル・コンボ・セーブ）
```

## たし算クイズの中身
- **入力**: テンキー（タブレットで押せる／PCは数字キー＋Enterでも可）
- **レベル**: ①10までの たしざん ②くり上がり たし算（こたえ11〜18）③20まで ミックス
- **くり上がりヒント**: まちがえると「10のかたまり」で考える解き方を表示
- **デイリーチャレンジ**: その日は何回でも同じ10問（日付シード）
- **まちがい復習**: まちがえた式だけ もう一回
- **育成（共有 store）**: 
  - 📖 けいさんカードずかん（1〜9のたし算 全45まい。1回で図鑑入り→3回で★）
  - 🏅 メダル（あそぶ／全問正解／累計正解／コンボ／なおし／図鑑／マスター／デイリー）
  - 🐣 称号17段（けいさんみならい → … → けいさんの神さま）
  - 💾 きろくの ほぞん／よみこみ（JSON）

## 進捗データ
- localStorage キー `sansuQuiz.v1`（オリジン単位で共有）。
- ひき算など別クイズを足しても、同じ store に合算される設計（漢字クイズと同じ方式）。

## これから足せるもの（トップに「じゅんびちゅう」で表示中）
- ➖ ひき算（くり下がり）／🔀 たし算ひき算ミックス／⏱️ タイムアタック
- ひき算の図鑑カードは `shared/sansu-core.js` の COLLECTIBLE を広げて対応。
